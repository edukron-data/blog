All Topics

  1. Pipeline Design Principles

    Requirements gathering, SLAs, and architecture decisions.

  2. Batch ETL Pipeline

    Designing batch ETL with ADF and Databricks.

  3. Incremental Load Patterns

    Watermark columns, change tracking, and MERGE.

  4. Streaming Pipeline

    Real-time ingestion with Event Hubs and Spark Streaming.

  5. Lambda Architecture

    Batch and speed layers for hybrid processing.

  6. Kappa Architecture

    Stream-only processing with replay capability.

  7. Data Quality Framework

    Validation rules, quarantine zones, and alerting.

  8. Pipeline Orchestration

    ADF triggers, Databricks workflows, and dependency management.

  9. Error Handling

    Dead letter queues, retry logic, and failure notifications.

  10. Monitoring & Alerting

    Pipeline health dashboards and SLA monitoring.

  11. Security in Pipelines

    Managed identity, encryption, and access control.

  12. Cost Management

    Right-sizing, scheduling, and storage optimization.

  13. Testing Pipelines

    Integration tests, data reconciliation, and smoke tests.

  14. CI/CD for Pipelines

    Automated deployment of ADF and Databricks artifacts.

  15. Documentation & Lineage

    Data dictionaries, pipeline docs, and lineage tools.

  16. Project: Retail Analytics

    End-to-end retail data pipeline from POS to dashboards.

  17. Project: IoT Telemetry

    Streaming IoT data ingestion and real-time analytics.

  18. Project: Financial Reporting

    Regulatory reporting pipeline with audit trails.

  19. Project: Healthcare Data

    HIPAA-aware pipeline with PHI handling patterns.

  20. Project: Legacy Migration

    On-premises to Azure cloud migration pipeline.

  21. Capstone: Full Pipeline

    Design, build, and document a production-grade data pipeline.
