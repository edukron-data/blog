Phase 2: Intermediate

All Topics

  1. Synapse Overview

    Unified analytics: SQL, Spark, pipelines, and studio.

  2. Synapse Studio

    Data hub, develop hub, monitor hub, and manage hub.

  3. Dedicated SQL Pools

    MPP architecture, distributions, and table design.

  4. Distribution Design

    Hash, round-robin, replicated tables, and skew.

  5. Indexing in Synapse

    Clustered columnstore, heap, and index strategies.

  6. Serverless SQL Pool

    On-demand querying over data lake files with OPENROWSET.

  7. External Tables

    CETAS, external tables, and lakehouse querying patterns.

  8. Spark Pools

    Synapse Spark, notebooks, and lakehouse processing.

  9. Synapse Pipelines

    Orchestration, data flows, and integration with ADF concepts.

  10. Linked Services

    Connecting to ADLS, SQL, and external data sources.

  11. Synapse Security

    Workspace access, SQL permissions, and managed identity.

  12. Monitoring Synapse

    SQL requests, Spark jobs, and pipeline runs.

  13. Performance Tuning

    Workload management, resource classes, and caching.

  14. Migration to Synapse

    From SQL DW, on-prem, and other warehouses.

  15. Synapse Capstone

    Build a lakehouse query pipeline with serverless and dedicated pools.
