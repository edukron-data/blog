All Topics

  1. Azure SQL Overview

    PaaS SQL options: single database, elastic pool, managed instance.

  2. Provisioning

    Creating databases, tiers, and compute/storage sizing.

  3. Connecting to Azure SQL

    Connection strings, firewall rules, and private link.

  4. Security Basics

    Authentication, encryption TDE, and Always Encrypted.

  5. Firewall & VNet Rules

    IP rules, service endpoints, and private endpoints.

  6. Performance Tiers

    DTU vs vCore models, serverless, and hyperscale.

  7. Query Store

    Query performance tracking, plan regression, and tuning.

  8. Indexing

    Index design, missing index recommendations, and maintenance.

  9. Elastic Pools

    Multi-tenant resource sharing and eDTU management.

  10. Geo-Replication

    Active geo-replication, failover groups, and DR planning.

  11. Backup & Restore

    Automated backups, point-in-time restore, and LTR.

  12. Migration Assessment

    DMA, Azure Migrate, and compatibility assessment.

  13. Migration Methods

    BACPAC, DMS, transaction replication, and cutover strategies.

  14. Serverless SQL

    Auto-pause, auto-scale, and cost-effective dev/test workloads.

  15. Synapse Link

    Near real-time analytics from Azure SQL to Synapse.

  16. Monitoring

    Metrics, alerts, Intelligent Insights, and diagnostics.

  17. Performance Tuning

    Wait statistics, blocking, and query optimization.

  18. Azure SQL Capstone

    Provision, secure, and tune an Azure SQL database.
