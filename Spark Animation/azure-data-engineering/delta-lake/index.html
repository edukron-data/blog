    JSON commit files, checkpointing, and log compaction.

    Schema on write, evolution, and mergeSchema option.

  9. Z-Ordering

    Multi-dimensional clustering for query performance.

  10. VACUUM

    Removing old files, retention policies, and storage cleanup.

  11. Change Data Feed

    CDC streams from Delta tables for downstream consumers.

  12. Delta Constraints

    CHECK constraints and data quality enforcement.

  13. Streaming & Delta

    Structured Streaming sink and source with Delta Lake.

  14. Unity Catalog & Delta

    Managed tables, sharing, and governance.

  15. Delta Performance

    Statistics, data skipping, and query optimization.

  16. Migration to Delta

    Converting Parquet/Hive tables to Delta format.

  17. Delta Lake Capstone

    Implement SCD Type 2 with MERGE and time travel.
