Path Phases

Phase 1: Foundations

Core concepts, terminology, and first hands-on exercises.

Phase 2: Intermediate

Applied patterns, integration, and real-world scenarios.

Phase 3: Advanced

Production tuning, architecture, capstone, and interview readiness.

Prerequisites

  • Willingness to learn sequentially
  • Dev environment setup for hands-on examples

What You Will Achieve

  • ✓ Apply Azure Data Factory concepts in practical projects
  • ✓ Understand architecture and production trade-offs
  • ✓ Build portfolio-ready artifacts from capstone steps

All Topics

  1. ADF Overview

    Cloud ETL/ELT service, integration runtime, and pipeline concepts.

  2. Linked Services

    Connection strings, authentication, and supported connectors.

  3. Datasets

    Dataset types, parameters, and schema definition.

  4. Pipeline Basics

    Activities, dependencies, and pipeline runs.

  5. Copy Activity

    Source-to-sink data movement, mapping, and performance.

  6. Mapping Data Flows

    Visual ETL transformations without coding.

  7. Data Flow Transformations

    Derived column, aggregate, join, and pivot in data flows.

  8. Schedule Triggers

    Time-based pipeline execution and timezone handling.

  9. Tumbling Window Triggers

    Batch processing windows and dependency chains.

  10. Event Triggers

    Blob events, custom events, and event-driven pipelines.

  11. Parameters & Variables

    Pipeline parameters, variables, and dynamic expressions.

  12. Integration Runtime

    Azure IR, self-hosted IR, and Azure-SSIS IR.

  13. Self-Hosted IR

    On-premises connectivity, HA setup, and troubleshooting.

  14. ADF Expressions

    Dynamic content, functions, and pipeline logic.

  15. Lookup Activity

    Reference data retrieval and foreach loops.

  16. ForEach Activity

    Parallel iteration over arrays and batch processing.

  17. Stored Procedure Activity

    Executing SQL stored procedures from ADF.

  18. Databricks Activity

    Running notebooks and jobs from ADF pipelines.

  19. Error Handling

    Try-catch scopes, retry policies, and failure notifications.

  20. Monitoring ADF

    Pipeline runs, alerts, and Azure Monitor integration.

  21. Git Integration

    Azure DevOps/GitHub integration and CI/CD for ADF.

  22. CI/CD for ADF

    ARM template export, publish branch, and automated deployment.

  23. ADF Performance

    Parallel copies, DIU tuning, and partition options.

  24. ADF Security

    Managed identity, Key Vault, and private endpoints.

  25. ADF Cost Optimization

    Pipeline efficiency, IR sizing, and activity optimization.

  26. Copy to Delta Lake

    Landing data in Delta format from ADF copy activity.

  27. Metadata-Driven Pipelines

    Dynamic pipelines driven by configuration tables.

  28. ADF Capstone

    Build an end-to-end ingestion pipeline with triggers and monitoring.
