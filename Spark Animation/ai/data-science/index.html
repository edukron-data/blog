Path Phases

Phase 1: Foundations

DS lifecycle, Python stack, stats, probability, distributions, hypothesis tests.

Phase 2: Analysis Skills

Cleaning, EDA, visualization, sampling, A/B tests, time series, SQL for analysts.

Phase 3: Professional Practice

Communication, ethics, tools, retail/churn case studies, portfolio, career path.

Prerequisites

  • Basic Python
  • High-school statistics helpful but not required

What You Will Achieve

  • ✓ Run rigorous EDA and hypothesis tests
  • ✓ Build compelling visual narratives
  • ✓ Design A/B experiments
  • ✓ Complete case-study analyses for your portfolio

All Topics

  1. What is Data Science?

    The data science lifecycle, roles, and core competencies.

  2. Data Types & Sources

    Structured, semi-structured, unstructured data and acquisition.

  3. Python for Data Science

    NumPy, pandas, matplotlib ecosystem overview.

  4. Jupyter Workflows

    Notebooks, reproducibility, and collaborative analysis.

  5. Descriptive Statistics

    Mean, median, mode, variance, and distribution shapes.

  6. Probability Basics

    Conditional probability, Bayes theorem, and random variables.

  7. Statistical Distributions

    Normal, binomial, Poisson, and when each applies.

  8. Hypothesis Testing

    p-values, t-tests, chi-square, and significance levels.

  9. Correlation & Covariance

    Pearson, Spearman, and interpreting relationships.

  10. Data Cleaning

    Missing values, outliers, duplicates, and imputation strategies.

  11. Feature Exploration

    Univariate and bivariate analysis techniques.

  12. Visualization Basics

    Bar, line, scatter, histogram charts with matplotlib/seaborn.

  13. Advanced Visualization

    Heatmaps, pair plots, and storytelling with data.

  14. EDA Workflow

    Systematic exploratory data analysis from raw to insights.

  15. Sampling Methods

    Random, stratified, bootstrap, and sampling bias.

  16. A/B Testing

    Experiment design, metrics, and statistical analysis of experiments.

  17. Time Series Intro

    Trends, seasonality, and basic time series decomposition.

  18. SQL for Analysts

    Analytical queries analysts use daily in warehouses.

  19. Communicating Insights

    Reports, dashboards, and stakeholder presentations.

  20. Ethics & Bias

    Fairness, privacy, and responsible data practices.

  21. Tools Ecosystem

    pandas, scikit-learn, SQL, and cloud notebook platforms.

  22. Case Study: Retail Analytics

    End-to-end analysis of sales and customer data.

  23. Case Study: Churn Analysis

    Identifying at-risk customers with EDA and modeling prep.

  24. Portfolio Projects

    Building a data science portfolio that gets interviews.

  25. Career Path

    Junior to senior data scientist skills and certifications.
