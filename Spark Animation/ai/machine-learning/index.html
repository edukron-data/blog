Path Phases

Phase 1: ML Basics

Paradigms, sklearn API, splits, linear/logistic regression, kNN, trees, forests, boosting.

Phase 2: Modeling Depth

SVM, Bayes, clustering, PCA, features, pipelines, CV, metrics, imbalance, tuning, selection, regularization.

Phase 3: Production ML

Feature selection, MLOps, deployment, tracking, time series, recommenders, anomalies, interpretability, capstone.

Prerequisites

  • Python and pandas
  • Basic statistics and linear algebra helpful

What You Will Achieve

  • ✓ Train and evaluate diverse sklearn models
  • ✓ Build leakage-safe pipelines with CV
  • ✓ Track experiments and deploy models
  • ✓ Deliver an end-to-end ML capstone

All Topics

  1. Machine Learning Overview

    Supervised, unsupervised, reinforcement learning paradigms.

  2. scikit-learn Setup

    Installation, API conventions, and estimator interface.

  3. Train/Test Split

    Holdout validation, random_state, and data leakage prevention.

  4. Linear Regression

    Ordinary least squares, coefficients, and R-squared.

  5. Logistic Regression

    Binary classification, sigmoid, and decision boundaries.

  6. K-Nearest Neighbors

    Instance-based learning, distance metrics, and k selection.

  7. Decision Trees

    Splitting criteria, tree depth, and interpretability.

  8. Random Forest

    Ensemble bagging, feature importance, and out-of-bag error.

  9. Gradient Boosting

    XGBoost, LightGBM concepts, and boosting vs bagging.

  10. Support Vector Machines

    Margins, kernels, and when SVMs excel.

  11. Naive Bayes

    Probabilistic classification for text and categorical data.

  12. K-Means Clustering

    Centroids, elbow method, and cluster evaluation.

  13. Hierarchical Clustering

    Agglomerative clustering and dendrograms.

  14. PCA & Dimensionality Reduction

    Principal components, variance explained, and t-SNE intro.

  15. Feature Engineering

    Encoding, scaling, binning, and domain features.

  16. ML Pipelines

    ColumnTransformer, Pipeline, and reproducible preprocessing.

  17. Cross-Validation

    K-fold, stratified CV, and nested cross-validation.

  18. Classification Metrics

    Accuracy, precision, recall, F1, ROC-AUC.

  19. Regression Metrics

    MSE, RMSE, MAE, and R-squared interpretation.

  20. Imbalanced Data

    SMOTE, class weights, and threshold tuning.

  21. Hyperparameter Tuning

    Grid search, random search, and Optuna intro.

  22. Model Selection

    Bias-variance tradeoff and choosing the right algorithm.

  23. Overfitting & Regularization

    L1/L2 regularization, dropout concept, and early stopping.

  24. Feature Selection

    Filter, wrapper, and embedded selection methods.

  25. MLOps Introduction

    Model versioning, deployment, and monitoring basics.

  26. Model Deployment

    Pickle, ONNX, REST APIs, and batch scoring.

  27. Experiment Tracking

    MLflow, metrics logging, and reproducibility.

  28. Time Series ML

    Lag features, rolling windows, and forecasting models.

  29. Recommender Systems

    Collaborative filtering and content-based approaches.

  30. Anomaly Detection

    Isolation Forest, statistical methods, and use cases.

  31. Model Interpretability

    SHAP, LIME, and explaining predictions to stakeholders.

  32. ML Capstone Project

    End-to-end classification or regression project.
