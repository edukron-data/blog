Learning Path Overview

The Agentic AI track teaches you to build autonomous systems that plan, use tools, collaborate, and operate reliably in production. The path moves from agent fundamentals through multi-agent orchestration, security, observability, and three real-world case studies ending in a capstone project.

Work through the topics below in order for the best experience — each lesson builds on the previous one from beginner through advanced.

Path Phases

Phase 1: Foundations

Agent architecture, ReAct, tool use, function calling, memory, planning, and self-reflection.

Phase 2: Multi-Agent Systems

Collaboration patterns, roles, communication, LangGraph/AutoGen/CrewAI, and human-in-the-loop.

Phase 3: Production & Case Studies

Evaluation, error recovery, security, cost tuning, deployment, observability, and capstone build.

Prerequisites

  • Completion of Generative AI (especially LLM agents and LangChain intro) is recommended
  • Comfort with Python and REST APIs
  • Familiarity with prompt engineering and RAG concepts

What You Will Achieve

  • ✓ Design and implement ReAct and plan-and-execute agent loops
  • ✓ Build multi-tool agents with LangGraph state machines
  • ✓ Evaluate agent trajectories and deploy with observability
  • ✓ Apply security controls and cost optimization in production
