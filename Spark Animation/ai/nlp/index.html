Path Phases

Phase 1: Text Foundations

Preprocessing, tokenization, n-grams, BoW, TF-IDF, embeddings.

Phase 2: Neural NLP

Sequence models, attention, transformers, BERT, GPT, classification, NER, POS, MT, QA.

Phase 3: Modern NLP

Summarization, spaCy, Hugging Face, prompting, evaluation, multilingual, production, ethics, RAG, capstone.

Prerequisites

  • Python
  • Basic ML concepts
  • Linear algebra basics helpful

What You Will Achieve

  • ✓ Build preprocessing and feature pipelines for text
  • ✓ Fine-tune transformer models for downstream tasks
  • ✓ Evaluate NLP systems with appropriate metrics
  • ✓ Ship a text classification or QA mini-project

All Topics

  1. NLP Overview

    Natural language processing tasks, challenges, and applications.

  2. Text Preprocessing

    Lowercasing, punctuation, stop words, and normalization.

  3. Tokenization

    Word tokenization, subword algorithms, and SentencePiece.

  4. Stemming & Lemmatization

    Reducing words to root forms with NLTK and spaCy.

  5. N-grams

    Character and word n-grams for language modeling features.

  6. Bag of Words

    CountVectorizer, document-term matrices, and sparsity.

  7. TF-IDF

    Term frequency-inverse document frequency weighting scheme.

  8. Word Embeddings

    Word2Vec, GloVe, and dense vector representations.

  9. Training Embeddings

    CBOW, Skip-gram, negative sampling, and vector arithmetic.

  10. Sequence Models Intro

    Why sequences need specialized architectures.

  11. RNNs for NLP

    Recurrent networks for tagging, classification, and generation.

  12. Attention Mechanism

    Query-key-value attention and its role in modern NLP.

  13. Transformers Introduction

    Self-attention, encoder-decoder, and positional encoding.

  14. BERT

    Masked language modeling, fine-tuning BERT for downstream tasks.

  15. GPT Models

    Autoregressive language models and generative text.

  16. Text Classification

    Sentiment analysis, topic classification, and fine-tuning.

  17. Named Entity Recognition

    Span labeling, BIO tagging, and NER models.

  18. Part-of-Speech Tagging

    Grammatical tagging with HMMs and neural taggers.

  19. Machine Translation

    Seq2seq, attention, and transformer translation systems.

  20. Question Answering

    Extractive and generative QA architectures.

  21. Text Summarization

    Extractive vs abstractive summarization approaches.

  22. spaCy Pipeline

    Tokenization, NER, and custom pipeline components.

  23. Hugging Face Intro

    Transformers library, model hub, and tokenizers.

  24. Prompting for NLP

    Zero-shot, few-shot, and instruction tuning for NLP tasks.

  25. NLP Evaluation

    BLEU, ROUGE, perplexity, and human evaluation.

  26. Multilingual NLP

    Cross-lingual models and translation for low-resource languages.

  27. NLP in Production

    Serving models, latency, batching, and monitoring.

  28. NLP Ethics

    Bias in language models, toxicity, and content moderation.

  29. RAG for NLP

    Retrieval-augmented generation for grounded text applications.

  30. NLP Capstone

    Build a text classification or QA mini-project.
