Path Phases

Phase 1: RNN Foundations

Sequence data, RNN architecture, forward pass, BPTT, LSTM intro.

Phase 2: Advanced RNNs

LSTM apps, GRU, bi-directional/stacked RNNs, seq2seq, teacher forcing, attention.

Phase 3: Applied RNNs

Time series, padding/masking, PyTorch/Keras, limitations, tuning, capstone.

Prerequisites

  • Neural network basics (ANN path helpful)
  • PyTorch or TensorFlow familiarity

What You Will Achieve

  • ✓ Implement LSTM/GRU models for sequences
  • ✓ Build seq2seq models with attention
  • ✓ Diagnose RNN training issues
  • ✓ Complete an LSTM capstone project

All Topics

  1. Sequences Introduction

    Sequential data types: text, time series, audio, and video.

  2. RNN Architecture

    Recurrent connections, hidden state, and unrolling through time.

  3. Forward Pass

    Computing outputs step-by-step across sequence length.

  4. Backprop Through Time

    Gradient flow through unrolled RNN and vanishing gradients.

  5. LSTM Architecture

    Forget, input, output gates, and cell state mechanics.

  6. LSTM Applications

    Language modeling, sentiment, and sequence prediction.

  7. GRU Architecture

    Reset and update gates — lighter alternative to LSTM.

  8. GRU vs LSTM

    Trade-offs in parameters, training speed, and performance.

  9. Bidirectional RNNs

    Forward and backward passes for context from both directions.

  10. Stacked RNNs

    Multi-layer recurrent networks for hierarchical features.

  11. Sequence-to-Sequence

    Encoder-decoder architecture for translation and summarization.

  12. Teacher Forcing

    Training seq2seq models with ground-truth vs predicted inputs.

  13. Attention in RNNs

    Bahdanau attention connecting encoder and decoder.

  14. Time Series with RNNs

    Forecasting, windowing, and multivariate sequences.

  15. Padding & Masking

    Handling variable-length sequences in batches.

  16. RNN in PyTorch

    nn.RNN, nn.LSTM, nn.GRU modules and training loops.

  17. RNN in Keras

    Sequential models with LSTM/GRU layers.

  18. RNN Limitations

    Long-range dependencies and why transformers emerged.

  19. RNN Hyperparameter Tuning

    Hidden size, layers, dropout, and learning rate.

  20. RNN Capstone

    Train an LSTM for text generation or time series forecasting.
