Path Phases

Phase 1: Network Basics

Neurons, perceptron, activations, MLPs, forward pass, loss, backprop, gradient descent.

Phase 2: Training Craft

Learning rate, optimizers, init, batch norm, dropout, overfitting, hyperparameter search.

Phase 3: Applied ANNs

PyTorch/Keras builds, classification/regression, visualization, debugging, capstone.

Prerequisites

  • Python
  • Basic calculus and linear algebra

What You Will Achieve

  • ✓ Derive and implement backprop for MLPs
  • ✓ Train stable networks with modern optimizers
  • ✓ Debug common NN training failures
  • ✓ Complete a feedforward network capstone

All Topics

  1. Biological Inspiration

    How artificial neurons relate to biological neurons.

  2. The Perceptron

    Single-layer network, linear separability, and limitations.

  3. Activation Functions

    Sigmoid, tanh, ReLU, Leaky ReLU, and softmax.

  4. Multi-Layer Networks

    Hidden layers, universal approximation, and depth.

  5. Forward Propagation

    Matrix operations through layers to compute outputs.

  6. Loss Functions

    MSE, cross-entropy, and choosing the right loss.

  7. Backpropagation

    Chain rule, gradient computation, and weight updates.

  8. Gradient Descent

    Batch, mini-batch, and stochastic gradient descent.

  9. Learning Rate

    Fixed, decay, and adaptive learning rate schedules.

  10. Optimizers

    SGD, Momentum, Adam, RMSprop, and when to use each.

  11. Weight Initialization

    Xavier, He initialization, and avoiding dead neurons.

  12. Batch Normalization

    Normalizing activations for faster, stable training.

  13. Dropout Regularization

    Random neuron dropout to prevent overfitting.

  14. Overfitting in ANNs

    Early stopping, L2 regularization, and data augmentation.

  15. Hyperparameter Search

    Grid search and random search for network design.

  16. Building ANNs in PyTorch

    nn.Module, Linear layers, and training loops.

  17. Building ANNs in Keras

    Sequential API, compile, fit, and evaluate.

  18. Classification with ANNs

    Softmax output, categorical cross-entropy, and metrics.

  19. Regression with ANNs

    Linear output layer and regression-specific considerations.

  20. Visualizing Training

    Loss curves, accuracy plots, and TensorBoard basics.

  21. Debugging Neural Networks

    NaN gradients, dead ReLUs, and common training failures.

  22. ANN Capstone

    Train a feedforward network on a tabular or MNIST dataset.
