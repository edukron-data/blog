Path Phases

Phase 1: CNN Foundations

Motivation, convolution, padding/stride, pooling, architecture, feature maps, classic architectures.

Phase 2: Training CNNs

ResNet, transfer learning, augmentation, classification, detection/segmentation intro, YOLO.

Phase 3: Production CV

PyTorch/Keras, regularization, interpretability, deployment, capstone fine-tuning.

Prerequisites

  • ANN fundamentals
  • PyTorch or TensorFlow basics

What You Will Achieve

  • ✓ Design and train CNN classifiers
  • ✓ Fine-tune pretrained models
  • ✓ Apply augmentation and regularization
  • ✓ Deploy or export models via ONNX

All Topics

  1. Why CNNs?

    Spatial hierarchies in images and parameter sharing benefits.

  2. Convolution Operation

    Filters, kernels, stride, and feature map computation.

  3. Padding & Stride

    Same vs valid padding and output dimension formulas.

  4. Pooling Layers

    Max pooling, average pooling, and spatial downsampling.

  5. CNN Architecture

    Conv-pool stacks, flatten, and fully connected classifier.

  6. Feature Maps

    What filters learn at each layer — edges to objects.

  7. Popular Architectures

    LeNet, AlexNet, VGG, ResNet overview.

  8. Residual Connections

    Skip connections, vanishing gradients, and ResNet blocks.

  9. Transfer Learning

    Pre-trained models, fine-tuning, and feature extraction.

  10. Data Augmentation

    Rotation, flip, crop, and synthetic training data.

  11. Image Classification

    Training and evaluating CNN classifiers end-to-end.

  12. Object Detection Intro

    Bounding boxes, IoU, and detection vs classification.

  13. YOLO Introduction

    Single-shot detectors and real-time detection concepts.

  14. Segmentation Intro

    Semantic vs instance segmentation overview.

  15. CNNs in PyTorch

    nn.Conv2d, nn.MaxPool2d, and torchvision models.

  16. CNNs in Keras

    Conv2D, MaxPooling2D, and pretrained applications.

  17. CNN Regularization

    Dropout, batch norm, and weight decay for CNNs.

  18. CNN Interpretability

    Grad-CAM, filter visualization, and saliency maps.

  19. Deploying CNN Models

    ONNX, TensorRT, and edge deployment considerations.

  20. CNN Capstone

    Fine-tune a pretrained model for a custom image dataset.
