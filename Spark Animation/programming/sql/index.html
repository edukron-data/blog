Path Phases

Phase 1: Query Foundations

SELECT, filtering, sorting, aggregates, GROUP BY, joins.

Phase 2: Advanced SQL

Subqueries, CTEs, windows, types, constraints, normalization, indexes, views, procedures, transactions.

Phase 3: Production SQL

Plans, tuning, temp tables, pivot, JSON, MERGE, security, analytical patterns.

Prerequisites

  • Comfort with tables and rows conceptually
  • Access to any SQL database (Azure SQL, PostgreSQL, or SQLite)

What You Will Achieve

  • ✓ Author complex analytical queries with joins and window functions
  • ✓ Read execution plans and tune slow queries
  • ✓ Design schemas with constraints and appropriate normalization
  • ✓ Apply MERGE/upsert and security best practices

All Topics

  1. Introduction to SQL

    Relational databases, SQL dialects, and the query execution model.

  2. SELECT Basics

    Retrieving columns and rows from tables with SELECT.

  3. WHERE & Filtering

    Filtering rows with comparison, IN, BETWEEN, and LIKE.

  4. ORDER BY & LIMIT

    Sorting results and limiting row counts.

  5. Aggregate Functions

    COUNT, SUM, AVG, MIN, MAX, and NULL handling.

  6. GROUP BY & HAVING

    Grouping rows and filtering aggregated results.

  7. INNER JOIN

    Combining tables on matching keys with INNER JOIN.

  8. OUTER JOINs

    LEFT, RIGHT, and FULL outer joins for incomplete matches.

  9. CROSS JOIN & Self Join

    Cartesian products and joining a table to itself.

  10. Subqueries

    Nested SELECT statements in WHERE, FROM, and SELECT clauses.

  11. Correlated Subqueries

    Subqueries that reference outer query columns.

  12. Common Table Expressions

    WITH clauses for readable, modular query design.

  13. Window Functions Intro

    OVER clause, PARTITION BY, and ROW_NUMBER.

  14. Ranking Functions

    RANK, DENSE_RANK, and NTILE for ordered analytics.

  15. LAG & LEAD

    Accessing previous and next row values in result sets.

  16. SQL Data Types

    Numeric, string, date/time, and JSON types across dialects.

  17. Constraints

    PRIMARY KEY, FOREIGN KEY, UNIQUE, CHECK, and NOT NULL.

  18. Normalization

    1NF through 3NF, denormalization trade-offs, and schema design.

  19. Indexes

    B-tree indexes, composite indexes, and covering indexes.

  20. Views

    Virtual tables, updatable views, and security through views.

  21. Stored Procedures

    Server-side procedures, functions, and parameters.

  22. Transactions & ACID

    BEGIN, COMMIT, ROLLBACK, and isolation levels.

  23. Query Execution Plans

    Reading EXPLAIN output and identifying bottlenecks.

  24. Performance Tuning

    Statistics, index selection, and query rewriting.

  25. Temporary Tables

    Session temp tables, table variables, and staging patterns.

  26. PIVOT & UNPIVOT

    Transforming rows to columns and back for reporting.

  27. JSON in SQL

    JSON columns, extraction operators, and semi-structured data.

  28. MERGE / Upsert

    INSERT ON CONFLICT, MERGE statements, and SCD patterns.

  29. SQL Security

    SQL injection prevention, grants, roles, and row-level security.

  30. Analytical SQL Patterns

    Running totals, top-N per group, and gap detection.
