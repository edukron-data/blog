Path Phases

Phase 1: Core Language

Setup, types, collections, control flow, and functions.

Phase 2: Intermediate Python

Scope, comprehensions, I/O, exceptions, modules, OOP, decorators, generators.

Phase 3: Production Skills

venv, packaging, NumPy/pandas, asyncio, pytest, typing, logging, best practices, capstone.

Prerequisites

  • Basic computer literacy
  • Text editor or VS Code
  • Optional: prior exposure to any programming language

What You Will Achieve

  • ✓ Write idiomatic Python 3 with proper structure and error handling
  • ✓ Use pandas/NumPy for data tasks and pytest for automated tests
  • ✓ Apply OOP, decorators, and async patterns in real scripts
  • ✓ Ship a small capstone project demonstrating path mastery

    Naming variables, assignment rules, and dynamic typing in Python.

  4. Data Types Overview

  5. Numbers & Math

    Integers, floats, arithmetic operators, and math module basics.

  6. Strings

    String creation, slicing, formatting, and common string methods.

  7. Lists

    Mutable sequences, indexing, slicing, and list methods.

  8. Tuples

    Immutable sequences, packing/unpacking, and when to use tuples.

  9. Sets

    Unique collections, set operations, and membership testing.

  10. Dictionaries

    Key-value mappings, dict methods, and nested structures.

  11. Operators

    Arithmetic, comparison, logical, and assignment operators.

  12. If Statements

    Conditional logic with if, elif, else, and truthiness.

  13. Loops

    for and while loops, break, continue, and loop patterns.

  14. Functions Basics

    Defining functions, calling functions, and docstrings.

  15. Function Parameters

    List, dict, and set comprehensions for concise transformations.

  18. File I/O

  19. Exception Handling

  20. Modules & Imports

  21. Classes & Objects

  22. Inheritance & Polymorphism

  24. Generators

  26. Virtual Environments

  27. pip & Packaging

  28. NumPy Introduction

    ndarrays, vectorization, and numerical computing fundamentals.

  29. Pandas Introduction

    DataFrames, Series, and essential data manipulation.

  30. Asyncio Basics

    async/await, event loops, and concurrent I/O patterns.

  31. Testing with pytest

    Writing tests, fixtures, parametrize, and TDD workflows.

  32. Type Hints

    Annotations, typing module, and static analysis with mypy.

  33. Logging

    logging module, log levels, handlers, and production logging patterns.

  34. Best Practices

    PEP 8, clean code, and production-ready Python habits.

  35. Capstone Project

    Build a small CLI or data script combining concepts from this path.
