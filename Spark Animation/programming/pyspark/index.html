Path Phases

Phase 1: Spark Foundations

SparkSession, RDDs, DataFrames, I/O, select/filter, aggregations.

Phase 2: Intermediate Spark

Joins, lazy/eager ops, SQL, UDFs, partitioning, caching, broadcast, skew, windows, streaming.

Phase 3: Production PySpark

Tuning, config, debugging, testing, Delta, patterns, architecture deep dive, capstone.

Prerequisites

  • Python fundamentals
  • Basic SQL
  • Optional: cluster computing concepts

What You Will Achieve

  • ✓ Build and optimize Spark DataFrame pipelines
  • ✓ Diagnose shuffle skew and OOM failures
  • ✓ Implement Structured Streaming and Delta Lake basics
  • ✓ Design an end-to-end batch ETL capstone

All Topics

  1. Introduction to PySpark

    Distributed computing, Spark ecosystem, and when to use Spark.

  2. SparkSession Setup

    Creating SparkSession, configs, and local vs cluster modes.

  3. RDD Basics

    Resilient Distributed Datasets — Spark's foundational abstraction.

  4. RDD Transformations

    map, filter, flatMap, and lazy evaluation on RDDs.

  5. RDD Actions

    collect, count, reduce, and triggering job execution.

  6. DataFrames Introduction

    Structured data, schemas, and Spark SQL integration.

  7. Reading Data

    CSV, JSON, Parquet, and JDBC sources with options.

  8. Writing Data

    Save modes, partitioning on write, and file formats.

  9. Select & Filter

    Column selection, where clauses, and column expressions.

  10. Aggregations

    groupBy, agg, pivot, and rollup/cube operations.

  11. DataFrame Joins

    Join types, broadcast hints, and join optimization.

  12. Lazy Transformations

    Transformation lineage, DAG, and catalyst involvement.

  13. Eager Actions

    show, count, collect, and understanding job triggers.

  14. Spark SQL

    spark.sql(), temp views, and global temp views.

  15. User-Defined Functions

    Python UDFs, Pandas UDFs, and performance trade-offs.

  16. Partitioning

    repartition, coalesce, partition pruning, and shuffle impact.

  17. Caching & Persistence

    cache, persist, storage levels, and memory management.

  18. Broadcast Joins

    When to broadcast, spark.sql.autoBroadcastJoinThreshold.

  19. Handling Data Skew

    Salting, AQE skew join, and repartition strategies.

  20. Window Functions

    Window specs, ranking, and analytic functions in Spark.

  21. Structured Streaming Intro

    Streaming DataFrames, triggers, and output modes.

  22. Watermarks & Late Data

    Event-time processing and stateful aggregations.

  23. Performance Tuning

    Spark UI, stages, tasks, and configuration knobs.

  24. Spark Configuration

    executor memory, cores, shuffle partitions, and AQE.

  25. Debugging Spark Jobs

    Reading logs, common exceptions, and troubleshooting.

  26. Testing PySpark

    Local testing, chispa, and unit test patterns.

  27. Delta Lake Intro

    ACID tables, time travel, and merge operations overview.

  28. Production Patterns

    Job scheduling, monitoring, and deployment on clusters.

  29. Spark Architecture Deep Dive

    Cluster architecture, scheduling, shuffle, and Catalyst optimizer.

  30. PySpark Best Practices

    Code organization, avoiding UDFs, and performance habits.

  31. Capstone ETL Job

    End-to-end batch pipeline design with PySpark.
