Path Phases

Phase 1: Container Basics

Concepts, install, images, Dockerfile, run, registry.

Phase 2: Intermediate Docker

Layers, volumes, networking, Compose, multi-stage builds, .dockerignore.

Phase 3: Production Docker

Security, scanning, ACR, CI integration, troubleshooting, capstone.

Prerequisites

  • Linux shell basics
  • Application you can containerize

What You Will Achieve

  • ✓ Write efficient multi-stage Dockerfiles
  • ✓ Orchestrate local stacks with Compose
  • ✓ Push images to ACR and integrate with CI
