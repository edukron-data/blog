Path Phases

Phase 1: Cloud Basics

Cloud models, regions, subscriptions, resource groups, portal, CLI, PowerShell.

Phase 2: Core Services

Compute, storage, networking, identity, RBAC, cost, monitoring.

Phase 3: Architecture & Career

Well-Architected Framework, ARM, Bicep, marketplace, certifications.

Prerequisites

  • General IT literacy
  • Free Azure account for hands-on labs

What You Will Achieve

  • ✓ Navigate Azure Portal and CLI confidently
  • ✓ Explain core compute/storage/network services
  • ✓ Apply RBAC and cost management basics
  • ✓ Prepare for AZ-900-level certification

All Topics

  1. Cloud Computing Overview

    On-premises vs cloud, benefits, and shared responsibility model.

  2. IaaS, PaaS, SaaS

    Service models and choosing the right Azure offering.

  3. Azure Regions & Zones

    Geography, availability zones, and region pairing.

  4. Subscriptions & Management Groups

    Organizing Azure resources and billing boundaries.

  5. Resource Groups

    Logical containers, tags, and lifecycle management.

  6. Azure Portal

    Navigating the portal, resource blades, and Cloud Shell.

  7. Azure CLI Basics

    Installing az CLI, login, and common commands.

  8. Azure PowerShell

    Connect-AzAccount and cmdlet-based management.

  9. Compute Services

    VMs, App Service, AKS, Functions, and Container Instances.

  10. Storage Services

    Blob, File, Queue, Table, and Disk storage options.

  11. Networking Overview

    VNets, subnets, NSGs, and Azure DNS.

  12. Identity Overview

    Microsoft Entra ID, users, groups, and applications.

  13. RBAC Basics

    Roles, scopes, and least-privilege access in Azure.

  14. Cost Management

    Pricing calculator, budgets, alerts, and reservations.

  15. Monitoring Overview

    Azure Monitor, Log Analytics, and Application Insights.

  16. Well-Architected Framework

    Five pillars: reliability, security, cost, operations, performance.

  17. ARM Templates Intro

    Declarative infrastructure as code with JSON templates.

  18. Bicep Introduction

    Simplified ARM syntax and deployment workflows.

  19. Azure Marketplace

    Third-party solutions and SaaS offerings in Azure.

  20. Certification Path

    AZ-900, AZ-104, and role-based certification roadmap.
