Path Phases

Phase 1: K8s Foundations

Architecture, kubectl, pods, replicas, deployments, services, namespaces.

Phase 2: Workload Design

ConfigMaps, secrets, volumes, ingress, labels, probes, resources, scheduling.

Phase 3: Production K8s

Helm, monitoring, security, troubleshooting, patterns, capstone manifest set.

Prerequisites

  • Docker fundamentals
  • YAML syntax

What You Will Achieve

  • ✓ Deploy and troubleshoot Kubernetes workloads
  • ✓ Configure services, ingress, and config/secrets
  • ✓ Apply resource limits and health probes

All Topics

  1. Kubernetes Overview

    Orchestration, control plane, and worker node architecture.

  2. Cluster Architecture

    API server, etcd, scheduler, controller manager, kubelet.

  3. kubectl Basics

    Config, get, describe, apply, and delete commands.

  4. Pods

    Pod lifecycle, multi-container pods, and restart policies.

  5. Deployments

    ReplicaSets, rolling updates, and rollback.

  6. Services

    ClusterIP, NodePort, LoadBalancer, and service discovery.

  7. ConfigMaps

    Configuration data injection via env vars and volumes.

  8. Secrets

    Secret management, base64 encoding, and external secret operators.

  9. Ingress

    Ingress controllers, rules, TLS, and path-based routing.

  10. Namespaces

    Resource isolation, quotas, and multi-tenancy.

  11. Labels & Selectors

    Labeling resources and selector-based operations.

  12. Resource Limits

    Requests, limits, QoS classes, and resource quotas.

  13. Health Probes

    Liveness, readiness, and startup probes.

  14. Persistent Volumes

    PV, PVC, StorageClass, and dynamic provisioning.

  15. StatefulSets

    Stable network identity and ordered deployment for stateful apps.

  16. DaemonSets

    Running pods on every node for logging and monitoring agents.

  17. Jobs & CronJobs

    Batch workloads and scheduled tasks in Kubernetes.

  18. Helm Introduction

    Charts, releases, and package management for K8s.

  19. Network Policies

    Pod-to-pod traffic control and micro-segmentation.

  20. Troubleshooting K8s

    Common failure modes, events, and debugging workflows.

  21. Kubernetes Capstone

    Deploy a multi-tier app with Deployment, Service, and Ingress.
