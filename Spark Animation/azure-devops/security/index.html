Path Phases

Phase 1: Security Foundations

Shared responsibility, Zero Trust, identity, RBAC deep dive, Key Vault, managed identity.

Phase 2: Infrastructure Security

NSGs, private endpoints, Defender, policy, secrets in pipelines.

Phase 3: DevSecOps & Compliance

SAST/DAST, supply chain, compliance (SOC2/GDPR), incident response, capstone hardening.

Prerequisites

  • Azure fundamentals
  • Basic networking concepts

    Secure score, recommendations, and threat protection.

  9. Microsoft Sentinel Intro

    SIEM, SOAR, and security analytics on Azure.

  10. DevSecOps

    Shift-left security, pipeline gates, and secure SDLC.

  11. SAST & DAST

  13. Compliance

  14. Identity Governance

  15. Incident Response

  16. Security Capstone
