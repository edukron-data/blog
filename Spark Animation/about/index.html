About EDUKRON BLOB

A programming blog for engineers who build and operate data systems at scale.

Mission

EDUKRON BLOB exists to make distributed systems and data engineering concepts accessible through clear writing, visual diagrams, and interactive demos. We believe the best way to understand Spark isn't memorizing API calls — it's seeing how a 30 GB job actually flows through a cluster.

Topics We Cover

  • Apache Spark — architecture, optimization, shuffle, memory, and deployment
  • Distributed Systems — fault tolerance, consistency, and scheduling
  • Data Engineering — pipelines, ETL patterns, and storage formats
  • Programming — Python, SQL, and the tools that glue it all together

Built With

This site is a static HTML/CSS/JS project designed for GitHub Pages. No build step, no framework — just fast, portable pages that work anywhere.
